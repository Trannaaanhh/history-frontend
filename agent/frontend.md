# Design Constraints

Use an ultra-clean, strict light-mode aesthetic that establishes immediate academic credibility.

The interface should feel like a premium educational platform used by top Vietnamese high schools and academic institutions.

Prioritize:

* Crisp typography
* Strong visual hierarchy
* Generous whitespace
* Consistent spacing system
* Subtle borders instead of shadows
* Professional card layouts
* Structured content presentation
* Excellent readability for long-form historical content

Avoid:

* Dark mode
* Glassmorphism
* Neumorphism
* Heavy drop shadows
* Cheap startup aesthetics
* Oversaturated colors
* Generic SaaS dashboards
* Excessive animations
* Loud gradients

Cards should primarily use:

* white backgrounds
* border border-red-100
* border border-amber-100
* shadow-sm only when necessary

Layout should emphasize:

* academic credibility
* historical authority
* trustworthiness
* focus on learning

The visual design should feel closer to a modern digital textbook and educational institution website than a startup landing page.

All pages must maintain visual consistency and use the same spacing, typography, and color system.

# Typography Rules

Headings:

* font-bold
* red-800
* strong hierarchy

Body text:

* neutral-700
* high readability
* comfortable line-height

Historical content:

* optimized for long reading sessions
* clear section separation
* subtle dividers between sections

# Component Styling Rules

Navigation:

* sticky
* white background
* bottom border
* active item highlighted in red

Cards:

* rounded-xl
* border
* minimal shadow

Buttons:

* clear hierarchy
* academic styling
* red / amber palette only

Charts:

* simple
* clean
* readable
* no flashy effects

The final result should look like a real educational platform ready for deployment by a Vietnamese Ministry of Education initiative or a premium national learning program.
